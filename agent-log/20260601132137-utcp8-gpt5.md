- 将 Electron 项目读取改为逐文件容错：成功文件继续返回，失败路径汇总为 `missingPaths`。
- Renderer 在项目恢复时打开可用文件，并在 notice 中显示跳过的缺失文件数量。
- 明确浏览器网页不能仅凭 JSON 中的绝对路径静默读取任意本地文件，但可通过 File System Access API 授权项目目录。
